# 🔎 UserScope

UserScope is a fast Username OSINT tool that searches for usernames across multiple social media, developer, gaming, blogging, and community platforms.

The tool helps investigators, bug bounty hunters, cybersecurity researchers, and OSINT enthusiasts discover public profiles associated with a username.

---

## ✨ Features

- Multi-platform username search
- Fast concurrent scanning
- Colored terminal output
- TXT report export
- JSON report export
- CSV report export
- Easy to extend with new websites

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/MoazMohamed891/userscope.git
cd Info_Name
```

Install requirements:

```bash
pip install requests
```

---

## 🚀 Usage

Run:

```bash
python userscope.py
```

Enter username:

```text
[?] Username: johndoe
```

---

## 📁 Reports

The tool automatically generates:

### TXT Report

```text
johndoe_results.txt
```

Contains only discovered profiles.

### JSON Report

```text
johndoe_results.json
```

Contains all scan results.

### CSV Report

```text
johndoe_results.csv
```

Contains structured results for spreadsheet analysis.
